# Adjust Volume - Alfred 2 Workflow

## Usage

**keyword:** ``volume``

**arguments:**
- ``max`` (set volume to 10)
- ``loud`` (7)
- ``half`` (5)
- ``medium`` (3)
- ``quiet`` (1)
- ``mute``, ``off`` (0)
- ``unmute`` (volume prior to mute)
- ``1``-``10``
